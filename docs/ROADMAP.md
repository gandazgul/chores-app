# Roadmap

- User authentication via Google Sign-In.
- Gotify notifications for chores.
- Displaying a list of chores.
- Adding new chores.
- Marking chores as done.
- Deleting chores.
- Displaying chore descriptions.
- Data persistence using a local SQLite database.
- Recurring chores functionality.
- Fuzzy search for chores.
- Advanced features for recurring chores
- Error handling and validation.
- Testing (unit, integration, e2e).
- Deployment setup. Likely containerized with docker, deployed to my home server.
